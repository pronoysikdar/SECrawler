# **App Name**: SECrawler

## Core Features:

- URL Input: Accept a URL as input from the user.
- Content Scraping: Scrape the content of the SEC filing from the provided URL.
- AI Summarization: Use an AI model to summarize the contents of the SEC filing once scraped, and display it to the user.
- Content Display: Display the scraped content to the user.
- Loading Indicator: Display a progress indicator while scraping and processing.

## Style Guidelines:

- Primary color: White or light grey for a clean background.
- Secondary color: Dark grey for text and important elements.
- Accent: Teal (#008080) for interactive elements and highlights.
- Clean and simple layout with clear sections for input, processing, and display.
- Use a clear and readable sans-serif font for the displayed text.
- Subtle loading animations during content scraping and processing.

## Original User Request:
I want to create an app which allows the user to enter an SEC url, which would point to an HTML page containing an SEC filing made by a US listed company. And then scrapes the contents of the URL and display it underneath in plain text format.
